# UI/UX 디자인 시스템 및 와이어프레임

**문서명:** MedAdReview UI/UX 디자인 시스템
**버전:** 1.1
**작성일:** 2026년 1월 7일
**참조 문서:** PRD v1.0, API 정의서 v1.0

---

## 1. 기술 스택

### 1.1 프론트엔드 기술 스택

| 기술 | 버전 | 용도 |
|------|------|------|
| **Next.js** | 16.x | React 프레임워크 (App Router) |
| **TypeScript** | 5.x | 타입 안정성 |
| **Tailwind CSS** | 4.x | 유틸리티 기반 스타일링 |
| **React** | 19.x | UI 라이브러리 |

### 1.2 주요 라이브러리

| 라이브러리 | 용도 |
|------------|------|
| @tanstack/react-query | 서버 상태 관리, 캐싱 |
| zustand | 클라이언트 상태 관리 |
| react-hook-form | 폼 관리 |
| zod | 스키마 검증 |
| recharts | 차트/데이터 시각화 |
| lucide-react | 아이콘 |
| date-fns | 날짜 처리 |

### 1.3 디렉토리 구조

```
src/
├── app/                    # Next.js App Router
│   ├── (auth)/            # 인증 관련 라우트
│   ├── (dashboard)/       # 대시보드 라우트
│   └── api/               # API Routes (BFF)
├── components/
│   ├── ui/                # 공통 UI 컴포넌트
│   ├── layout/            # 레이아웃 컴포넌트
│   ├── review/            # 심의 관련 컴포넌트
│   └── dashboard/         # 대시보드 컴포넌트
├── hooks/                  # 커스텀 훅
├── stores/                 # Zustand 스토어
├── services/               # API 서비스
├── types/                  # TypeScript 타입
└── lib/                    # 유틸리티
```

## 5. 컴포넌트

### 5.1 판정 배지

```
┌──────────┐  ┌──────────────┐  ┌──────────┐  ┌──────────┐
│ ✓ 허용   │  │ ⚠ 조건부허용 │  │ ✕ 불허   │  │ ○ 보류   │
└──────────┘  └──────────────┘  └──────────┘  └──────────┘
 success       warning           error          neutral
```

### 5.2 심각도 배지

```
┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
│ CRITICAL │  │   HIGH   │  │  MEDIUM  │  │   LOW    │
└──────────┘  └──────────┘  └──────────┘  └──────────┘
```

### 5.3 버튼

| 타입 | 용도 | Tailwind 클래스 |
|------|------|-----------------|
| **Primary** | 주요 액션 | `bg-emerald-500 hover:bg-emerald-600 text-white` |
| **Secondary** | 보조 액션 | `border border-gray-300 bg-white hover:bg-gray-50 text-gray-700` |
| **Danger** | 위험 액션 | `bg-red-500 hover:bg-red-600 text-white` |
| **Ghost** | 텍스트 링크 | `text-emerald-600 hover:text-emerald-700 hover:bg-emerald-50` |
| **Outline** | 테두리 버튼 | `border border-emerald-500 text-emerald-600 hover:bg-emerald-50` |

### 5.4 버튼 크기

| 크기 | Tailwind 클래스 |
|------|-----------------|
| Small | `px-3 py-1.5 text-sm` |
| Medium | `px-4 py-2 text-base` |
| Large | `px-6 py-3 text-lg` |

### 5.5 입력 필드

```tsx
// 기본 입력 필드 스타일
<input className="
  w-full px-3 py-2
  bg-white border border-gray-300 rounded-lg
  text-gray-900 placeholder-gray-400
  focus:outline-none focus:ring-2 focus:ring-emerald-500 focus:border-transparent
  disabled:bg-gray-100 disabled:text-gray-500
" />
```
